- [Angular Style Guide](https://angular.io/guide/styleguide#lift)

1. **L**ocating our code is easy
2. **I**dentify code at a glance
3. **F**lat structure as long as we can
4. **T**ry to stay DRY (Don't Repeat Yourself) or T-DRY
